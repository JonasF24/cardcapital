# Card Capital - Quick Start

## Option 1: Run Dashboard Locally (5 Minutes)
1. Put `card-tracker-dashboard.html` in a local folder.
2. Add logo assets in the same folder:
   - `cardcapital-logo.png`
   - `cardcapital-icon.png`
3. Open `card-tracker-dashboard.html` in a browser.

## Option 2: Deploy Backend (30 Minutes)
1. Open `card-tracker-scraper/DEPLOYMENT_GUIDE.md`.
2. Configure `.env` from `.env.example`.
3. Install dependencies and run `python run_pipeline.py`.
4. Add GitHub Actions secrets and push.

## Option 3: Full Integration (45 Minutes)
- Complete Option 1 + Option 2, then wire dashboard data to Supabase/API.
